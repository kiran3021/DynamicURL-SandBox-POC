There are two ways to use the iframe element:

  1. To embed content from another page
  2. To embed user-generated content

Embedding content from another page

Use the src attribute to specify the URL of the page to embed:

&lt;iframe src="https://example.org"&gt;&lt;/iframe&gt;

Embedding user-generated content

Use the srcdoc attribute to specify the content to embed. This post is already an example!
